Motivation

We can get insight into the Einstein's equations by considering the traditional initial value (Cauchy) problem. Consider that we are give \(g_{\mu\nu}\) and \(\partial g_{\mu\nu}/\partial x^0\) everywhere on the "plane" \(x^0=t\). If we could find from this the second partial derivatives \( \partial^2 g_{\mu\nu}/\partial (x^0)^2\) at some \(x^0 = t\) we could then compute \(g_{\mu\nu}\) everywhere.

Spacetime foliation -- needs a graph

We assume that we have such a space-time, that is foliated by a smooth family of space-like hypersurfaces \(\Sigma_t\), where \(t\) denotes some continuous parameter
  • The notion of hypersurface is some (n-1) smooth submanifold
  • Space-time foliation means that every event in such a space-time belongs to exactly one hypersurface \(\Sigma_t\)
We choose \(n^\mu\) to be future oriented unit normal to \(\Sigma_t\). Since each hypersurface corresponds to \(t=const\) the normal has to read \[n_\alpha := -N \frac{\partial t}{\partial x^\alpha},\] where \(N\) is the normalization factor usually called the lapse. We also introduce the tangent field to the direction of time \[t^\mu = \frac{\partial x^\mu}{t}.\label{eqt}\] Clearly it follows \[n_\alpha t^\alpha = N.\] If we introduce the projection of \(t^\mu\) on \(\Sigma_t\) (usually called the shift) \[N^\mu := (\delta^\mu _\alpha + n^\mu n_\mu)t^\alpha =: {h^\mu}_\alpha t^\alpha .\] \(h_{\mu\nu}\) acts as the metric of \(\Sigma_t\).

We now consider a congruence of curves \(\gamma\) intersecting the hypersurface \(\Sigma_t\) and we use \(t = t(x^\alpha)\) and parameter of these curves. By this construction the \eqref{eqt} is always tangent to the congruence. A particular curve (\gamma_p\) defines a mapping from a point \(P\) on \(\Sigma_t\) to a point \(P'\) on \(\Sigma_t'\) and further on for all \(t\). To fix the coordinates on \(P'\) and \(P''\), given some intrinsic coordinates \(y^\alpha(P)\) on \(\Sigma_t\), we simply impose \(y^\alpha(P'')=y^\alpha(P')=y^\alpha(P)\). figure 4.2 poisson
This construction defines a coordinate system \((t,y^\alpha)\). From the originaly used system \(x^\alpha = x^\alpha(t,y^a)\) From here we can write \[dx^\mu = \frac{\partial x^\mu}{\partial t}dt + \frac {\partial x^\mu}{\partial y^m}dy^m.\] Rewriting \(t^\mu = N n^\mu + N^\mu \) we get \[ds^2 = -N^2 dt^2 + h_{m n}(N^m dt + dy^m)(N^n dt + dy^n).\] and by Sylvester's determinant identity \[\sqrt{-g} = N \sqrt{h}\] Lastly if we have adapted coordinates \(x^0 = t\) and \(x^i\) covering \(\Sigma_t\), in which case we have \(x^i = \delta^i_m y^m\).

Extrinsic curvature

Our motivation for the foliation was solving the Cauchy problem, but we assumed to basically known our space-time already, since we assumed to know the foliation everywhere. It is better to consider the evolution of the spatial geometry \(h_{ik}\) on a certain hypersurface $\Sigma$. So now we describe the intrinsic geometry of such a hypersurface by the metric \(h_{ik}(t)\), such metric is also called the first fundamental form. If we want to compute the of \(h_{ik}(t)\) according to Einstein equations the inital conditions must also include the time derivative \(h_{ik,t}\), but that is clearly non-tensorial!
The suitable tensor turns out to be the expansion tensor \(\Theta_{\mu\nu}\) \[\Theta_{\mu\nu} = h^\alpha_\mu h^\beta_\nu n_{(\alpha ; \beta)}= = =\frac{1}{2} \mathcal{L}_n h_{\mu\nu}\] This part I don't get, something about symmetry but anyway it leads to \[K_{\mu \nu} := n_{\alpha;\beta} h^\alpha_\mu h^\beta_\nu = \frac{1}{2}\mathcal{L}_n h_{\mu\nu},\] which is called the extrinsic curvature or the second fundamental form. This form lives on \(\Sigma_t\) and so \[h_{mu\nu}n^nu = 0, \; \alpha_\mu n^\mu = 0, \; K_{\mu \nu}n^\nu = 0, \; K:= h^{\mu nu}K_{\mu\nu} = {n^\mu}_{;\nu} \label{eq:eq}\] Since \(h_{\mu \nu}\) describes the intrisic geometry of the hypersurface, \(K_{\mu\nu}\) describes the curvature of the hypersurface as a submanifold for a given manifold.
The definition of \(K_{\mu\nu}\) would be zero if the covector \(n_\mu\) would be parallel transported along \(h^\alpha_{\nu}\), hence it describes how much it points to different directions along \(\Sigma_t\). The trace \(K\) has also a clear geometrical sense: it is \(>0\) when \(\Sigma\) is convex and \( < 0 \) \(\Sigma\) is concave.

Covariant derivative on hypersurface

Projection by the \(h^\alpha_\beta\) tensor ensures a unique and geometry transition between the tensor living in a space-time and those linving on the hypersurface \(\Sigma\). But would we define a "gradient" that keeps such tensor "purely spatial"? We define a covariant derivative corresponding to the Levi-Civita connection of a the metric as \[ {T^{\mu\dots}}_{\alpha \dots | \rho} :={T^{\nu\dots}}_{\beta\dots;\sigma}h^\mu_\nu \dots h^\beta_\alpha \dots h^\sigma_\rho \] Levi-Civita connecting need to be a covariant derivate, needs "annihilate" the metric and has to have zero torsion. All of theese are easily proven.
